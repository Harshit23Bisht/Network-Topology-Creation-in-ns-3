Tools Used

ns-3 Network Simulator

NetAnim Visualization Tool

C++

How to Run

Place .cc files inside ns-3.44/scratch/

Run simulation using:

./ns3 run scratch/<filename>

Open generated .xml file in NetAnim to view topology.

Output

Each program generates an animation file showing the network structure and node connections.
